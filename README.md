# GardenKey MVP v11 — Pilot Polish

This version is intended as the first real-use pilot build.

## What is included

- Stable NFC URL structure: `plant.html?id=GK-LAV-001`
- Polished mobile plant profile page
- Dashboard/admin page
- Blank Add Plant workflow with placeholders
- Four sample plants in `plants.json`
- Pilot feedback section on each plant profile
- Feedback link on the admin dashboard

## How to deploy

Upload all files in this folder to the existing GitHub repository, replacing the current files. Commit directly to `main`.

## Test URLs

- `/plant.html?id=GK-LAV-001`
- `/plant.html?id=GK-ROS-001`
- `/admin.html`

## Pilot reminder

Use this version for a few days before adding new features. Capture what feels natural, what feels awkward, and what people ask for.
