How to test with NFC

  1. Upload this folder to GitHub Pages.
  2. Open one plant page on your phone.
  3. Copy the URL.
  4. Write that URL to an NFC tag using an Android NFC writer app.
  5. Tap the tag and confirm the page opens instantly.
