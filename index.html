Quick setup guide

1: Open NFC Tools

Tap Write → Add record → URL / URI.

2: Paste the GardenKey URL

Use a starter page, plant profile, pot page or tester create page.

3: Write to tag

Tap Write, approach the tag, and wait for Write Complete.

4: Test the tag

Close the app and scan the tag normally. It should open GardenKey.

Weekly GardenKey fact
