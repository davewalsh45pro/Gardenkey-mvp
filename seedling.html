Seedling & nursery mode

A lightweight tracker for seed trays and propagation batches — sowing through to transplant. Works for any batch of seedlings, from a single windowsill tray to a full nursery workflow. Saved to this browser; use Export to back it up or hand it off.

Trays / batches

Save to this browser keeps your trays only on this device/browser, so they're here next time you open the page. Export JSON downloads a file of all your trays — use it as a backup, or to move them to another device. Import JSON loads a previously exported file back in (e.g. one made on another device, or one someone else sent you).
