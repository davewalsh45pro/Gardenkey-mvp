Static GitHub Pages cannot save directly to your website. This editor lets you edit plant data, preview it locally, then download an updated plants.json file to upload to GitHub.

Workflow: Load plant → edit fields → save to browser preview → download JSON → upload it to data/plants.json in GitHub.
